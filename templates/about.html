About the project

A Machine Learning App

The project is a machine learning web application.

With given datasets about Disaster Responses messages and categories, the backend managed the data ETL pipelines engineering. Then generate a machine learning model based on the processed data.

Users can directly use the web app to call the model to estimate any new disaster responses messages.

General statistics overviews and visualizations are also provided.

The backend developed based on Python, Flask Framework, Pandas, Scikit-Learn, NLTK, ...

The frontend developed based on Bootstrap 5, Plotly.

Data layer uses SQLite.

Data engineering use a custom data process script based on the given csv.

Machine Learning model has been trained and dumped as a encoded file. Since it is serialized, it can be re-used by external app also.
